# pruebaDesempe-oJS
The file is downloaded and you need to install the JSON server
npm init -y 
and also the modules
npm install json-server --save-dev 
Then, in the terminal, start the JSON server with the command
json-server --watch db.json

The program allows you to register and log in as either a user or an administrator. After that, you can create a task, which you can edit or delete as needed, and change its status from pending to in progress and completed.
